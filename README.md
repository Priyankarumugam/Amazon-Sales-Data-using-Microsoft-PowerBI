# Amazon-Sales-Data-using-Microsoft-PowerBI
Data Visualization of Amazon Sales Data
## 🎯 Project Objective
The goal of this dashboard is to analyze sales data to help stakeholders make data-driven decisions to increase profitability.

## 📊 Key Insights
- **Total Revenue:** $3.22M
- **Top Sales:** $1M .
- **Top Profit:** $3,08M(total revenue).

## 🛠️ Tools Used
- Power BI (Desktop)
- Data Source: Amazon Sales Dataset

## 🛠️ Features
- **Interactive Slicers:** Filter by Region, Category, and Date.
- **KPI Cards:** Display Total Revenue, Profit, and Quantity.
- **Charts:** Monthly Sales Trend, Profit by Customer.
